Key Bindings

  • Basic Gestures
    • Space / Tap Anywhere
      flips the card for the answer
    • Up Key / Swipe Up
      recognized the card
    • Down Key / Swipe Down
      didn't recognize the card
  • Buttons
    • Inbuilt
      to use already built flashcards
    • Import
      to import json flashcards from local device
    • Export
      to save unfinished flashcards to study later
    • Help?
      ...

Logic

  • When the card is recognized, it is removed from the current study deck.
  • When the card is unrecognized, that card will be called again after 30 seconds.
  • Unrecognizing that card will repeat the process again until it is recognized.
